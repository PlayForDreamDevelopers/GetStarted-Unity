## 关于项目

“入门”示例是一个简单的Unity项目，演示如何使用 YVR Unity SDK和 Unity XR 交互工具包创建一个简单的VR应用程序。您可以将此项目作为您自己的VR应用程序的起点。

## 要求

- Unity 2022.3.52f1 或者更新的版本
- Unity 包：
  - YVR Unity SDK 2.3.1 或者更新的版本
  - XR Interaction Toolkit 2.4.3 或者更新的版本
